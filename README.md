# Woodlife
使用 Vue CLI3 及 Vue Router 完成 SPA 互動式電商網站，涵蓋前後台，串接 API 實做出商品管理 (CRUD)、購物車、優惠券、登入介面等功能，並使用 ESLint Standard

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
